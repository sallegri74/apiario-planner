Il miele

Il processo

Come nasce il nostro miele

Dal volo delle api alla tua tavola, ogni passaggio è curato a mano.

1

Bottino

Le api raccolgono nettare in zone collinari ricche di flora spontanea.

2

Maturazione

Estraiamo il miele solo quando è perfettamente maturo nelle celle.

3

Lavorazione

Filtraggio a freddo per preservare polline, enzimi e profumi.

4

Invasamento

Riempimento manuale dei vasetti, etichettatura e controllo qualità.

L'apicoltore

Una passione che diventa miele

Conduciamo i nostri apiari nel cuore del Ticino, in zone collinari ricche di flora spontanea. Le nostre api bottinano in ambienti incontaminati, a contatto con boschi di castagno, acacia e tigli secolari.

Ogni vasetto è il risultato di un lavoro stagionale che rispetta i ritmi naturali delle colonie: ogni gesto, dalla cura dell'arnia all'etichettatura, è pensato per portare nel barattolo un piccolo pezzo del Ticino.
